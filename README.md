# AdventCalendarApp

1. Download this Github repository (should include main.py, myclasses.py and run.ipynb).
2. Install Jupyter notebook (https://jupyter.org/) or similar.
3. Open the run.ipynb in the notebook and run it. You need a working internet connection. It might take a few minutes for the program to run.
4. Repeat step 3 every day for a new message ! Enjoy !
